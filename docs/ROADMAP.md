# 🌙 Khwab Development Standard

1. Freeze architecture before implementation.
2. Complete one Task before starting the next.
3. Complete one Step before the next Step.
4. Inspect existing code before changing it.
5. Every Task must build successfully.
6. Every Phase ends with documentation.
7. Every Phase ends with one Git commit.
8. The Brain remains frozen.
9. Capabilities plug into the Brain through Skills.
10. No placeholder or unfinished code.
